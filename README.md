# GameCafe
The system represents a game caf√© where customers reserve tables, borrow games and order food and drinks. Employees manage reservations and assist customers. The database tracks customers, reservations, tables, games, orders and employees.
# Entities
- Customers - Stores customer information such as name, email and phone number.
- Cafe_Tables - Represents tables available in the cafe and their seating capacity.
- Reservations - Records reservations made by customers including data and time details.
- Games - Stores information about available board games.
- Games_Copies - Represents indivisual copies of games available for borrowing.
- Loans - Tracks game copies bowworwed during reservations and their return status.
